arbhar Noise Tests

Findings

Having arbhar present in a eurorack case induces audible noise at the output when it is powered.

  • Using the m05 Silencer didn't seem to improve the noise floor.
  • Physically moving arbhar outside of the case didn't improve the noise floor.
  • A simple subtractive patch included noise when arbhar is powered, even though arbhar wasn't involved in the patch.
  • The gain boosting I've used in these tests is loud enough to the point that I believe the noise is typically very low with a correctly gain-staged patch.
  • I didn't record this, but as arbhar boots you can hear it go from no noise, to noises that are different to its usual noise, until the Raspberry Pi finishes booting and the noise I've recorded here starts.

Based on these findings I believe that it's extremely important to use correct gain-staging when using arbhar in a patch. If possible arbhar should be housed in a separate case, otherwise all of your patches will have some additional noise even when the module is not used.

Also, because the noise doesn't seem to significantly change when moving arbhar outside of the case, it seems like it is a case of arbhar introducing noise into the power bus rather than RF interference.
